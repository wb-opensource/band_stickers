Privacy Policy

This privacy policy applies to the Chrome extension Band Mattermost Stickers. The extension adds a local sticker picker to Mattermost on https://band.wb.ru.

Data Collection

Personal information

The extension does not collect names, email addresses, account identifiers, passwords, payment data, or government identifiers.

Message content

The extension does not read, store, transmit, or analyze Mattermost message text. It only attaches the sticker file selected by the user.

Browsing activity

The extension is limited to https://band.wb.ru/* and does not monitor browsing activity on other websites.

Data Stored Locally

Band Mattermost Stickers stores a small local usage counter in the user's browser to build the Popular sticker tab. This data stays on the user's device and is not sent to the developer or to any third-party service.

Network Requests And Third Parties

The extension does not send data to developer-owned servers, analytics providers, advertising networks, or data brokers. Sticker assets are bundled with the extension and loaded through Chrome extension URLs. When a user sends a sticker, Mattermost handles the upload in the same way it handles a file manually attached by the user.

Permissions And Site Access

The extension runs only as a content script on https://band.wb.ru/*. It does not request broad Chrome permissions such as tabs, history, cookies, identity, storage, or host access to all websites.

Data Sharing

Band Mattermost Stickers does not sell, rent, transfer, or disclose user data. The extension does not use collected data for advertising, creditworthiness, profiling, or unrelated purposes.

Data Retention And Deletion

Local sticker usage statistics remain in the browser until the user clears site data for band.wb.ru, clears browser data, resets the browser profile, or removes the extension. Uninstalling the extension removes the bundled extension files from Chrome.

How to clear local usage data

In Chrome, open site settings or browsing data settings and clear local site data for band.wb.ru. The extension will rebuild the Popular tab from new sticker usage after that.

Children's Privacy

The extension is intended for workplace Mattermost use and is not directed to children. It does not knowingly collect personal information from children.

Changes To This Policy

If the extension changes in a way that affects data handling, this privacy policy will be updated before or together with the new extension release.

Contact

For privacy questions about Band Mattermost Stickers, use the support contact shown on the Chrome Web Store listing for this extension.
