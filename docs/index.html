Band Mattermost Stickers

Расширение добавляет компактный выбор стикеров в Mattermost на домене band.wb.ru, ставит кнопку между emoji и upload в основном композере и тредах, а выбранный стикер передает как PNG-вложение в активное поле.

Текущие стикерпаки

Подключены PUSI с 27 WebP/PNG, ASIA с 16 WebP, MONIKA с 17 WebP, Конкуренты с двумя WebP, PUPA с 38 WebP и Жириновский с пятью WebP. Первая вкладка Популярные строится из локальной статистики использования. Для каждого стикера есть маленький PNG 128x128 в stickers_128/. По умолчанию отправляется маленький файл; чекбокс bigstic включает отправку оригинального размера. Перед загрузкой WebP конвертируется в PNG. Во время прикрепления файла расширение кратко держит в активном композере невидимый символ U+200B, затем удаляет его и при включенной моментальной отправке нажимает кнопку отправки.

Автопроверки

В GitHub Actions добавлены проверки синтаксиса, валидности manifest, консистентности реестра стикеров и базовой безопасности расширения. Они защищают от пропущенных файлов, неверных 128 PNG, лишних permissions и расширения домена за пределы https://band.wb.ru/*.

Как добавить новый файл

Добавьте WebP или PNG в папку нужного пака, сгенерируйте PNG 128x128 в stickers_128/, затем зарегистрируйте path и smallPath в src/stickers.js и разрешите оба файла в manifest.json. Для технических папок используйте ASCII-имена, например stickers/ZHIRINOVSKIY/. Новый пак добавляется отдельным объектом в window.BAND_STICKER_PACKS.
