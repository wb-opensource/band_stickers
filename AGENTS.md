# AGENTS.md

## Проект

Chrome Extension Manifest V3 для добавления локальных стикеров в Mattermost на `https://band.wb.ru`.

## Правила разработки

- Расширение должно оставаться ограниченным доменом `https://band.wb.ru/*`.
- Не добавлять широкие разрешения Chrome без необходимости.
- При изменении списка стикеров обновлять `src/stickers.js`, `README.md`, `DESIGN.md` и `docs/index.html`.
- При изменении поведения UI или установки обновлять HTML-документацию.
- Стили держать изолированными через префикс `band-stickers`.
- Для новых стикерпаков использовать локальные файлы в `stickers/<PACK_NAME>/`.
- Для технических имён папок стикерпаков использовать ASCII, даже если видимое название пака кириллическое.
- Кнопка стикеров должна монтироваться в панели действий композера между emoji picker и загрузкой файлов.
- Попап стикеров должен рендериться через `document.body`, чтобы его не обрезал контейнер композера Mattermost.
- Открытие попапа должно работать через `pointerdown` и защищаться от перехвата события toolbar-обработчиками Mattermost.
- Горячая клавиша `Command + Option + S` должна открывать и закрывать попап стикеров.
- Каждый стикерпак должен отображаться как отдельная папка в панели выбора.
- Первой вкладкой должна быть `Популярные`, построенная из локальной статистики использования.
- WebP-стикеры должны отправляться в Mattermost как PNG после клиентской конвертации.
- Панель должна содержать чекбокс `Моментальная отправка`, включенный по умолчанию.
- Панель должна содержать чекбокс `bigstic`, выключенный по умолчанию; без него отправляется 128x128 PNG из `smallPath`.
- Для каждого стикера поддерживать оригинал в `stickers/` и маленький PNG 128x128 в `stickers_128/`.
- Подключать только WebP/PNG/JPG/JPEG; `.tgs` оставлять вне реестра до появления рендера Lottie.
- Во время прикрепления файла нужно кратко держать невидимый символ `U+200B` в композере Mattermost, затем удалить его до автоотправки.
- При добавлении файла стикера обновлять `manifest.json` в `web_accessible_resources`.

## Проверка

- Проверить валидность `manifest.json`.
- Загрузить расширение через `chrome://extensions` как `Load unpacked`.
- Проверить, что кнопка появляется только на `https://band.wb.ru`.
- Проверить, что кнопка стоит между emoji picker и загрузкой файлов.
- Проверить горячую клавишу `Command + Option + S`.
- Проверить выбор папки стикеров и добавление стикера в поле сообщения Mattermost.
- Проверить, что после отправки стикер появляется во вкладке `Популярные`.
- Проверить автоотправку с включенной `Моментальная отправка` и ручной режим с выключенной опцией.
- Проверить отправку маленького стикера по умолчанию и оригинала при включенном `bigstic`.
- Проверить отсутствие `ERR_FILE_NOT_FOUND` для каждого файла из `src/stickers.js`.

## Общие инструкции пользователя

Python: используй `tqdm` в процессах, не используй аргументы для запуска скриптов. Форматируй SQL с отступами и ключевыми словами в верхнем регистре. Работай с ClickHouse и PostgreSQL, добавляй комментарии там, где они нужны. Во всех примерах Python добавляй обработку ошибок `try/except` с логированием, если есть работа с сетью или файлами. В SQL-запросах предлагай оптимизации.

Старайся всегда генерировать HTML-документацию и заполнять `README.md` и `AGENTS.md`. HTML-документацию делай красивой и лаконичной, добавляй графики, схемы, интерактивные элементы. Всю документацию обновляй при каждом изменении кода, который влияет на эту документацию.
